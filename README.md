# Fix Dashes

Convert annoying em-dashes (—) and en-dashes (–) to regular hyphens (-) instantly.

## Usage

1. Select text with em-dashes or en-dashes
2. Run the "Fix Dashes in Selected Text" command
3. Your text is automatically replaced with fixed version

Perfect for cleaning up AI-generated text that uses long dashes!

## Example

**Before:** `Lorem Ipsum—placeholder text—remains standard`  
**After:** `Lorem Ipsum - placeholder text - remains standard`
